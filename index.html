The purpose of this project is to show the skills and knowledge I’ve gained throughout this course. It involves using various front-end web development techniques to build a functional and responsive website. Throughout the course, I’ve learned how to design websites, create layouts, and add interactive features. This project is a chance to put all those lessons into practice. Key features of this project include creating responsive grids, which allow the website to adjust to different screen sizes, making it look great on both desktop and mobile devices. Each of these technologies plays an important role in making the site look good, function smoothly, and be interactive. This project is the result of everything I’ve learned and shows my ability to create a complete, well-designed, and user-friendly website that works across different devices.

HOVER EFFECT

A hover effect is a visual or interactive change that occurs when the user moves their cursor over an element on a website, like an image or button. Hover effects can activate many visual changes, such as changing the background, opacity,including shadows,etc. The hover effect is really popular in web design to enhance user engagement by suggesting interactivity. By offering a visual response, hover effects make web development feel more engaging.

STEP 1

To customize an image hover effect, you must write your HTML code before anything else. First, you need to add in a div class called grid. Then create another div class="img-wrapper" inside of the div you have just created, along with img class="blur" src="(add image link)". Finally, insert another div class="content fade", and inside that, add whatever text you want to display.

STEP 2

Start with creating the class .grid. After, arrange the images into a grid using display:grid, and defining the columns with grid-template-columns. Afterwards, create your image-wrapper and experiment with its various structures. To place the wrapper, set the position to absolute and inset:0.

STEP 3

Add an image wrapper/content fade, and modify the opacity to your liking.
